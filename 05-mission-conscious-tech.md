# Fundamental Research: Computational Neuroscience

## Core Function

To apply rigorous metrics from neuroscience to evaluate machine learning models.

## Technical Approach

### Integrated Information Theory
We implement algorithms to calculate the "Phi" value of neural network layers. This measures the amount of information generated by the system as a whole above and beyond the sum of its parts.

### Global Workspace Theory Implementation
We are architecting "broadcast" modules within neural networks that simulate the global workspace, allowing distinct specialized sub-networks (vision, language, memory) to share information explicitly, rather than implicitly via gradient descent.

### Objective
To identify the "indicator properties" of machine consciousness by monitoring internal state changes during anomaly detection tasks.

## Tech Spec 1001

- **Methodology**: Perturbation analysis on activation maps
- **Validation**: Statistical correlation against human EEG/fMRI datasets during similar cognitive tasks

class GlobalWorkspace:
    def __init__(self):
        self.broadcast_channels = {}
    
    def broadcast(self, module_name, information):
        # Explicit information sharing between modules
        for channel in self.broadcast_channels:
            if channel != module_name:
